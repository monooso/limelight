# Limelight

A GNOME Shell extension that dims inactive windows, so the one you're working in stands out.

One setting. No clutter.

## Installation

### From source

```bash
git clone https://github.com/monooso/limelight.git
cp -r limelight ~/.local/share/gnome-shell/extensions/limelight@stephenlewis.me
```

Log out and back in (or restart GNOME Shell on X11), then enable:

```bash
gnome-extensions enable limelight@stephenlewis.me
```

## Configuration

Open the extension's preferences (via Extension Manager or GNOME Extensions) to set the opacity of inactive windows. That's it.

## Requirements

GNOME Shell 49+.
